# tqi_evolution_backend_2021
tqi evolution backend desafio.

### Descrição 
O projeto consiste em uma API, cuja entrada é dados de cliente, que solicita um determinado empréstimo, e saida é informações desse mesmo cliente e dos seus empréstimos solicitados. Sendo assim, opitei por utilizar a ferramenta spring security, para autenticação do usuário e administrador; o banco de dados, em mémória, h2; assim como outras dependências, ver pom.xml. É válido salientar, que utilizei, por opção, juros compostos na definição do montante a ser pago pelo cliente em um número, pré-definido, de parcelas e valor do emprestimo solicitado. 



1. Dependências:
- Spring Security;
- h2;
- Lombok;
